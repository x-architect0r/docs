## Setup your development

Learn how to set up your environment and start building with Buildr.

<AccordionGroup>
  <Accordion icon="wallet" title="Connect your Web3 wallet">
    Before diving in, you'll need a Web3 wallet:
    1. Install MetaMask or any BSC-compatible wallet
    2. Add Binance Smart Chain to your wallet
    3. Get some BNB for gas fees (0.01 BNB minimum recommended)
    
    <Note>
      Make sure to securely store your wallet recovery phrase!
    </Note>
  </Accordion>

  <Accordion icon="rocket" title="Access Buildr">
    Choose your preferred way to use Buildr:
    
    - **Telegram**: Connect with our Telegram bot
    - **Web App**: Access through our web interface
    - **X (Twitter)**: Interact via X messages
    
    <Warning>
      Always verify official links from our website to avoid scams.
    </Warning>
  </Accordion>
</AccordionGroup>

### Preview & Test

<AccordionGroup>
  <Accordion icon="flask" title="Try your first commands">
    Start with these simple commands:
    ```bash
    # Check your setup
    "Show my wallet balance"
    
    # View market trends
    "Show me trending tokens"
    
    # Make your first swap
    "Swap 0.1 BNB to PEPE"
    ```
    <Tip>
      Start with small amounts while getting familiar with the system.
    </Tip>
  </Accordion>

  <Accordion icon="shield" title="Verify & Secure">
    Before any transaction:
    1. Check transaction details in your wallet
    2. Verify token contracts on BSCScan
    3. Ensure sufficient BNB for gas
    4. Review permissions and approvals
  </Accordion>
</AccordionGroup>

## Best Practices

<AccordionGroup>
  <Accordion icon="check-double" title="Transaction Safety">
    - Always verify transaction details
    - Start with small test amounts
    - Keep enough BNB for gas
    - Check token contracts on BSCScan
  </Accordion>
  
  <Accordion icon="lightbulb" title="Pro Tips">
    - Use specific commands for better results
    - Monitor gas prices for better timing
    - Set reasonable slippage for volatile tokens

</AccordionGroup>

<Warning>
